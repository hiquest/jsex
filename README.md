# 💥 JSEX 💥
A collection of little and fun JavaScript exercises

## How to use it?

1. Clone the repo.
2. Run `npm run test` and watch the tests fail.
3. Fix the tests one by one.
4. 🍕💃🍕💃🍻🍻🍻
